# Dishuu

Dishuu is a parent company building resilient systems across communications, autonomy, fabrication, consulting, and developer infrastructure.

We work on technology for real-world environments, not ideal conditions.

## What Dishuu Is Building

- **Dishuu Consulting**  
  IT consulting, product delivery, modernization, architecture, and resilient systems execution.

- **Dishuu Zero**  
  Peer-to-peer and infrastructure-resilient communications.

- **Dishuu Aero**  
  Autonomous aerial systems and mobility for field operations.

- **Dishuu Forge**  
  Distributed fabrication and local hardware continuity.

- **Dishuu Dev**  
  APIs, software surfaces, and technical papers that support the wider Dishuu stack.
